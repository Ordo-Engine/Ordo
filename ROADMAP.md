## Quick Overview

| Milestone | Version | Goal |
|-----------|---------|------|
| **First Decision** | v0.5 | 5-minute onboarding with templates and guided setup |
| **Deploy & Connect** | v0.6 | One-click publish from Studio to Engine, SDK integration |
| **Observe** | v0.7 | Execution dashboard, trace explorer, alerting |
| **Govern** | v0.8 | Change requests, impact analysis, audit log |
| **Decision Topology** | v0.9 | Organization-wide decision service map |
| **Ordo Cloud** | v1.0 | Managed platform with hosted engine |

Milestones 1–5 are fully open source (MIT). Ordo Cloud adds managed hosting and enterprise features.
